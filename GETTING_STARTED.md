# Getting Started — Akademie Robotů / Robot Academy

## How to run the game in 3 steps

### Step 1 — Import into Unity
1. Open **Unity Hub** → New Project → **2D (Core)** template → name it anything → Create
2. Close Unity
3. Copy everything from this folder into your new Unity project folder (overwrite `Assets/`, `Packages/`, `ProjectSettings/`)
4. Reopen Unity — it will compile all scripts

### Step 2 — One-click setup
In Unity's menu bar:

```
VacuumVille → Setup Entire Project
```

This single click automatically:
- Creates all **21 scenes** (LanguageSelect, Home, LevelSelect, MathTask, LevelComplete, ParentDashboard + all 11 minigame scenes)
- Creates all **11 LevelDefinition** ScriptableObjects
- Creates all **9 CharacterDefinition** ScriptableObjects
- Generates **30 placeholder colored sprites** (circles/rounded rects) so the game is visually playable immediately
- Wires up **every serialized field** in every scene via reflection
- Registers all scenes in **Build Settings** in the correct order

### Step 3 — Press Play
Open `Assets/Scenes/LanguageSelect.unity` → press **Play**. The full game runs in Czech (or English after switching).

## Building for Android

1. In Unity: **File → Build Settings** → select **Android** → Switch Platform
2. Set minimum API to **24 (Android 7.0)**
3. **Player Settings → Other Settings:**
   - Package name: `com.yourname.akademieroboту`
   - Orientation: Portrait only
   - Scripting Backend: IL2CPP
   - Target Architectures: ARM64
4. Connect Android device with USB debugging enabled
5. **Build and Run**

│       ├── cs-CZ/plurals.json   ← Czech plural forms
│       ├── en-US/strings.json   ← English UI strings
│       └── en-US/plurals.json   ← English plural forms

ProjectSettings/
│   └── ProjectSettings.asset    ← Android portrait, API 24+, IL2CPP
